# 🏥 Portal Hospital ALI

Este é um portal web interativo desenvolvido em **React** com **Vite**, focado no gerenciamento de informações institucionais, novidades e agendamentos para o **Hospital ALI**.

O projeto foi estruturado de forma colaborativa e conta com um sistema de rotas dinâmicas para navegação entre as páginas criadas pelos integrantes do grupo.

---

## 🛠️ Tecnologias Utilizadas

* **React** (Biblioteca para construção da interface)
* **Vite** (Ferramenta de build e servidor de desenvolvimento ultra rápido)
* **React Router DOM** (Gerenciamento de rotas e navegação do portal)
* **CSS3** (Estilização responsiva e customizada)

---

## 🚀 Como Rodar o Projeto Localmente

Siga os passos abaixo para baixar as dependências e executar o servidor de desenvolvimento na sua máquina:

### 1. Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina:
* [Node.js](https://nodejs.org/) (Recomendado a versão LTS)
------------------------------------------------------------------------------------------------------------------------------------------
### 2. Instalar as Dependências
Abra o terminal do seu editor de código (como o VS Code) na raiz do projeto e execute o comando abaixo para instalar todas as bibliotecas necessárias (como o React Router):
```bash
npm install

------------------------------------------------------------------------------------------------------------------------------------------
3. Iniciar o Servidor de Desenvolvimento
Após o término da instalação, inicie o servidor local digitando: 
npm run dev
------------------------------------------------------------------------------------------------------------------------
4. Acessar o Portal
O terminal irá gerar um link local (geralmente http://localhost:5173/).
Basta segurar a tecla Ctrl e clicar no link, ou copiá-lo e colá-lo no seu navegador para visualizar o site funcionando.
